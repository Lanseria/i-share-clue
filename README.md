# 线索分享平台

## 技术栈

- POSTGis gis
- vue.js@3 framework
- naive-ui@2 ui
- postgres database
- nestjs backend
- AMap@2 gis-show
- minio file-save-server

## 目的

提供线索
互帮互助

## 进程

1. 简单权限设定
2. 简单用户信息设定
3. 线索位置添加
4. 线索位置搜索
